PACTOLS est un [[20210914180730]] thésaurus créé par [[20210914180338]] Frantiq dans le but d’organiser les mots-clés utilisés dans l’[[20210914180427]] indexation des ressources documentaires de domaine archéologique. Il couvre toutes les thématiques de l’archéologie, de la préhistoire à l’époque contemporaine. Les concepts sont classés dans sept thématiques : Peuples, Anthroponymes, Chronologie, Toponymes, Oeuvres, Lieux, Sujets (PACTOLS). 
Le thésaurus est interopérable et multilingue (7 langues), chaque concept possède un identifiant [[20210914180126]] ARK qui en fait un objet web unique, citable et réutilisable sur le [[20210914180814]] web sémantique.

Site web : <https://pactols.frantiq.fr/opentheso/>
